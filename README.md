# inertia-dotnet

The Inertia.js server-side adapter for ASP.NET Core.

This is a port of the official [Inertia.js](https://inertiajs.com) Laravel adapter to ASP.NET Core (.NET 10+). Rather than a 1:1 translation, InertiaCore is redesigned from the ground up to take advantage of C#'s type system, dependency injection, and async model.
